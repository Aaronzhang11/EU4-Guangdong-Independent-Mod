# Guangdong Independent Practice

《欧陆风云4》广东独立架空历史MOD。

## 当前版本

v0.3

## 主要内容

- 1444年广东独立开局
- 控制现代广东、海南和澳门
- 广东专属国家理念
- 专属君主、继承人与初始顾问
- 初始11k陆军与8艘舰船
- 广东专属开局背景
- 与明朝保持特殊宗属关系和十年停战

## 兼容版本

- Europa Universalis IV v1.37.*
- 当前版本面向中文汉化环境
- 不保证兼容修改中国地图、开局历史或开局介绍的其他MOD

## 安装方法

将以下两个项目放入：

`文档\Paradox Interactive\Europa Universalis IV\mod`

- `guangdong_independent_practice.mod`
- `guangdong_independent_practice`文件夹

随后在Paradox启动器中启用MOD，并开始一个新的1444年游戏。(只打开该mod和两个中文翻译包)
